Description du projet

L’agence souhaite publier la nouvelle version de son site vitrine. Il s’agit d’un site one-page.

L'HISTOIRE

Le design a été validé, et un premier développeur freelance a commencé l’intégration il y a quelques semaines.
Malheureusement, ce freelance a dû abandonner le projet pour raisons personnelles.
Le site est fonctionnel, mais quelques bugs entravent le bon usage par les visiteurs. 724events a donc fait appel à moi pour finaliser le travail.
J'ai également rencontré Jean-Baptiste, le directeur Marketing de l’agence, en visioconférence pour qu’il me présente plus précisément le projet.

MON APPROCHE

Dans ce projet, j'ai dû débugger et finaliser le développement d'un site one-page.
J'ai utilisé les Chrome DevTools pour identifier et comprendre les problèmes et bugs signalés dans le site web actuel.
Après avoir identifié les problèmes, j'étais chargée de résoudre ces bugs et problèmes. Cela impliquera l'utilisation de mes compétences en JavaScript, ainsi que des outils spécifiques à React, comme React Developer Tools.
J'ai également dû compléter le cahier de recette. Ce document détaillera tous les scénarios de test nécessaires pour vérifier le bon fonctionnement du site, incluant des tests unitaires et fonctionnels.
Pour la gestion du projet, j'ai utilisé Yarn comme gestionnaire de packages et Node.js pour exécuter du code JavaScript côté serveur.
Ces outils m'ont aidé à gérer les dépendances et à exécuter des tests localement. Enfin, j'ai versionné mon travail avec GitHub et utilisé Visual Studio Code comme éditeur de code pour développer et débugger le site.
